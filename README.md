# Booklore 工程说明

## 最近更新（2026-03-20）

### PDF格式化功能 V2.1 - 双页检测与进度优化
- ✅ **新增双页智能检测算法**：基于宽高比 + 中心亮度差异的双重判定机制
  - 宽高比 < 1.30：直接判定为单页
  - 宽高比 >= 1.34：强判定为双页（即使无明显装订线）
  - 中间区域亮度比 >= 1.15：判定为双页（检测装订线）
- ✅ **新增双页自动切分**：检测到双页后自动切分为左右两页分别格式化
- ✅ **优化进度推送机制**：
  - 新增 sub_stage 字段，细化子阶段描述
  - 新增统计字段：双页数、单页数、错误数实时显示
  - 改进进度计算逻辑，确保单调递增，避免进度条跳跃
- ✅ **前端UI优化**：
  - 进度条、统计数字、详细日志分离显示
  - 日志持久化（localStorage，5分钟有效期）
  - 日志区域无边框、无滚动条、无数量限制
  - 修复统计数据归零问题（避免 undefined 重置计数器）
- ✅ **修复编译问题**：解决 Angular 编译器无法解析模板字符串中中文的问题

详见：`print-engine/README.md`
