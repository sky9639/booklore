# Booklore 工程说明

## 最近更新（2026-03-25）

### 拼版工作台稳定性与 AI 裁切流程收敛
- ✅ **Workspace 单一写入入口**：`workspace.json` 统一由 print-engine 持久化，减少多端并发写入带来的覆盖风险
- ✅ **Gemini 展开图 / 裁切流程稳定化**：

- ✅ **裁切弹窗交互优化**：
  - 支持滚轮缩放、拖拽平移、键盘微调裁切线
  - 预览刷新增加过期保护，避免快速操作时显示旧结果
  - 初始缩放更贴合可视区域，减少手动调整成本
- ✅ **拼版工作台状态管理收敛**：
  - 前端 workspace 状态统一经由状态服务发布
  - 参数修改、素材切换后自动清理失效 PDF 状态
  - 历史预览与当前草稿恢复逻辑统一
- ✅ **元数据链路鲁棒性增强**：
  - 中断信号传播更完整，避免后台任务被吞掉
  - GoodReads 等元数据源降级为 fallback，降低主流程阻塞风险

## 核心目录说明
- **booklore-api/**：Spring Boot 后端源码（Java 17）
- **booklore-ui/**：Angular 前端源码（Angular 19）
- **print-engine/**：Python 打印引擎源码（PDF格式化 + AI生成）
- **shared/**：共享数据目录（图书文件、素材、生成结果）

## 核心功能模块

### 1. 图书管理
- 图书导入（PDF电子书）
- 图书元数据管理（书名、作者、页数等）
- 图书删除（含关联文件清理）
- PDF阅读器（支持原始PDF预览）

### 2. 拼版工作台
- **成书参数配置**：开本尺寸（A4/A5/B5）、页数、纸张厚度
- **素材管理**：封面/书脊/封底上传、选择、删除、历史记录
- **AI生成 / 裁切**：
  - Gemini 展开图生成（封面+书脊+封底一体化）
  - 左侧展开图历史列表 + 当前草稿恢复
  - 可视化裁切线调整、缩放、拖拽、实时预览
  - 裁切保存后自动回填 front_output / spine / back
  - 多组配置管理（联通参数 + 提示词模板）
- **拼版预览**：实时预览印刷展开图效果
- **PDF生成**：生成可直接打印的PDF文件

### 3. PDF格式化
- **智能检测**：双页识别（宽高比 + 中心亮度差异）
- **自动处理**：双页切分、内容边界检测、非等比拉伸
- **尺寸支持**：A4/A5/B5标准尺寸
- **安全机制**：自动备份、恢复功能
- **进度监控**：实时进度推送、统计数据展示

## 技术栈
- **前端**：Angular 19 + TypeScript + SCSS
- **后端**：Spring Boot 3 + Java 17
- **AI引擎**：Python 3.11 + FastAPI + Gemini API
- **图像处理**：Pillow + PyMuPDF
- **通信**：RESTful API + SSE（Server-Sent Events）
