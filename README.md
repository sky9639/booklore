# Booklore 工程说明

## 最近更新（2026-03-28）

### 拼版工作台日志中心化与 Gemini 配置链路增强
- ✅ **操作日志重构**：拼版工作台右侧由“生成日志”升级为统一“操作日志”，集中记录参数变化、PDF 格式化、AI 裁切、AI 生图等关键动作
- ✅ **日志鲁棒性提升**：新增日志类型抽象、状态着色、条数上限、滚动节流与参数日志去抖，避免刷屏、卡顿与长会话下日志无限增长
- ✅ **左侧语义修正**：原“PDF尺寸”区块更名为“PDF 信息与格式化”，使标题与当前功能（尺寸识别、格式化按钮、进度与局部日志）保持一致
- ✅ **AI 裁切流程可追踪**：打开裁切、恢复草稿、生成展开图、保存裁切、删除历史等动作均纳入统一日志，便于回溯当前工作台状态
- ✅ **AI 配置弹窗交互加固**：禁止点击遮罩直接关闭；存在未保存修改时，关闭前需确认，避免配置误丢失
- ✅ **Gemini 能力动态检测**：新增 `imageSize` 支持探测与能力指纹缓存，模型保持可自定义；仅在测试确认支持时开放高分辨率选项
- ✅ **AI 生图兼容性增强**：统一 Gemini 图片请求体格式，按能力决定是否发送 `imageSize`，兼容不同 Gemini 版本与中转网关
- ✅ **错误提示更清晰**：AI 生图失败改为结构化错误分类，前端展示更明确的中文失败原因，减少网关异常排查成本
- ✅ **展开图历史稳定性修复**：修复 AI 生图进行中删除历史项后被旧 workspace 重新带回前端的问题，避免历史区出现“有记录无图片”的异常状态
- ✅ **前端构建验证通过**：本轮 `print-workspace` / Gemini 配置 / print-engine 改动已完成 Angular 构建检查

详见：`print-engine/README.md`

## 核心目录说明
- **booklore-api/**：Spring Boot 后端源码（Java 17）
- **booklore-ui/**：Angular 前端源码（Angular 19）
- **print-engine/**：Python 打印引擎源码（PDF格式化 + AI生成）
- **shared/**：共享数据目录（图书文件、素材、生成结果）

## 核心功能模块

### 1. 图书管理
- 图书导入（PDF电子书）
- 图书元数据管理（书名、作者、页数等）
- 图书删除（含关联文件清理）
- PDF阅读器（支持原始PDF预览）

### 2. 拼版工作台
- **成书参数配置**：开本尺寸（A4/A5/B5）、页数、纸张厚度
- **素材管理**：封面/书脊/封底上传、选择、删除、历史记录
- **AI生成 / 裁切**：
  - Gemini 展开图生成（封面+书脊+封底一体化）
  - 左侧展开图历史列表 + 当前草稿恢复
  - 可视化裁切线调整、缩放、拖拽、实时预览
  - 裁切保存后自动回填 front_output / spine / back
  - 多组配置管理（联通参数 + 提示词模板）
- **拼版预览**：实时预览印刷展开图效果
- **PDF生成**：生成可直接打印的PDF文件

### 3. PDF格式化
- **智能检测**：双页识别（宽高比 + 中心亮度差异）
- **自动处理**：双页切分、内容边界检测、非等比拉伸
- **尺寸支持**：A4/A5/B5标准尺寸
- **安全机制**：自动备份、恢复功能
- **进度监控**：实时进度推送、统计数据展示

## 技术栈
- **前端**：Angular 19 + TypeScript + SCSS
- **后端**：Spring Boot 3 + Java 17
- **AI引擎**：Python 3.11 + FastAPI + Gemini API
- **图像处理**：Pillow + PyMuPDF
- **通信**：RESTful API + SSE（Server-Sent Events）
