📋 拼版说明
01
上传素材
在右侧「素材准备」区分别上传封面、书脊、封底图片。建议使用 JPG / PNG 格式，图片宽度 ≥ 3000px 以确保印刷清晰度。
02
设置成书参数
选择成书尺寸（A4 / A5 / B5），填写书籍页数与纸张厚度，系统将自动计算书脊宽度。
书脊公式 书脊宽度 = 页数 × 纸张厚度
03
检查拼版预览
右下方预览区将根据尺寸自动生成拼版效果。
A4 模式 封面 · 书脊 · 封底（3页独立）
A5 / B5 封面＋书脊拼合 · 封底（2页）
04
生成印刷文件
确认预览无误后，点击右侧「生成印刷PDF」按钮，系统将输出符合印刷规格的 PDF 文件。
